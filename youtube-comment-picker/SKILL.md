# YouTube 댓글 이벤트 선정

이벤트에 달린 댓글 중 **진짜 사람이 정성껏 쓴 댓글**을 골라낸다.
URL을 주면 댓글을 자동으로 수집하고, 직접 붙여넣어도 된다.

## 입력값

다음 중 하나로 입력한다:

- **YouTube URL** (예: `https://www.youtube.com/watch?v=XXXXX`) → 자동 수집
- **댓글 목록 직접 붙여넣기** → `닉네임: 댓글 내용` 형식

공통 옵션:
- (선택) 선정 인원 수 (기본값: 5명)
- (선택) 이벤트 조건 (예: "구독자 우선", "짧아도 진심이면 OK")

## URL로 입력받았을 때 — 댓글 자동 수집

Bash 도구로 아래 명령을 실행해 댓글을 가져온다.

```bash
/Users/hanwha/Library/Python/3.9/bin/yt-dlp \
  --write-comments \
  --skip-download \
  --no-warnings \
  -o "/private/tmp/claude-501/-Users-hanwha/f96da85d-6228-4653-8732-3e197191a974/scratchpad/yt_comments" \
  "[URL]"
```

생성된 `.info.json` 파일에서 `comments` 배열을 읽어 `author` + `text` + `id` 필드를 추출한다.
댓글이 너무 많으면 최신순 500개까지만 사용하고 사용자에게 알린다.
`id` 필드로 댓글 직접 링크를 생성한다: `https://www.youtube.com/watch?v=VIDEO_ID&lc=COMMENT_ID`

## 필터링 기준 — 이 댓글은 탈락

다음 중 하나라도 해당하면 **즉시 제외**하고 탈락 사유를 명시한다.

| 탈락 유형 | 판별 신호 |
|-----------|-----------|
| **AI 작성 의심** | 문장 구조가 지나치게 완벽·정제됨 / 형용사 남발 ("정말 유익하고 알찬 콘텐츠") / 구체적 개인 경험 없이 칭찬만 / 글머리 기호·번호 매기기 / 첫문장이 영상 제목 그대로 반복 |
| **복붙·도배** | 동일하거나 90% 이상 유사한 내용이 다른 닉네임으로 반복 |
| **성의 없음** | 3단어 이하 (예: "좋아요!", "최고에요", "ㅋㅋㅋ") / 이모지만 나열 / 영상과 무관한 내용 |
| **광고·스팸** | 외부 링크 포함 / 자기 채널 홍보 / 무관한 제품 언급 |

## 선정 기준 — 이 댓글이 통과

점수는 아래 4가지 기준 합산 (각 25점, 총 100점):

1. **개인 경험** — 나만의 구체적인 에피소드, 상황, 감정이 담겼는가
2. **영상 연결** — 영상 내용을 실제로 보고 반응한 흔적이 있는가 (특정 장면·발언 언급 등)
3. **진정성** — 꾸밈없는 구어체, 솔직한 감정 표현, 인간적 온기가 느껴지는가
4. **문장 완성도** — 맞춤법·분량과 무관하게 하고 싶은 말이 분명히 전달되는가

## 출력 형식

### 댓글 수집 결과 (URL 입력 시)

```
총 수집: N개 / 분석 대상: N개 (최신순)
```

### 탈락 댓글 요약

```
탈락 총 N건
- AI 의심: N건 (예: "정말 유익하고 알찬 콘텐츠였습니다...")
- 성의 없음: N건
- 복붙/도배: N건
- 광고/스팸: N건
```

### 최종 선정 댓글

통과한 댓글 중 점수 순으로 정렬해 선정 인원만큼 출력한다.

🔗 [댓글 바로가기](https://www.youtube.com/watch?v=VIDEO_ID&lc=COMMENT_ID)

> 댓글 원문 그대로

### 운영자 코멘트

- 통과 댓글이 선정 인원보다 적으면 경고: "정성 댓글이 N개뿐입니다. 선정 인원을 줄이거나 이벤트를 재공지하는 것을 권장합니다."
- AI 의심 비율이 30% 이상이면 경고: "AI 작성 의심 댓글이 많습니다. 댓글 이벤트 공지에 '본인 경험 필수' 조건 추가를 권장합니다."

## 작성 원칙

- **닉네임은 그대로** 표기한다. 익명 처리하지 않는다.
- 점수는 투명하게 공개해 이의 제기에 대비한다.
- AI 판별은 확정이 아닌 **"의심"** 으로 표현한다. "AI가 쓴 게 확실하다"고 단정하지 않는다.
- 짧아도 진심이 느껴지면 성의 없음으로 분류하지 않는다. 길이보다 내용이 우선이다.
- 한국어 댓글 특유의 구어체(ㅋㅋ, ㅠㅠ, ~요 생략 등)를 감점 요인으로 삼지 않는다.
